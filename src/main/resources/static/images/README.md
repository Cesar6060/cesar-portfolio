# Images Directory

## Required Images

Please add the following images to this directory:

### 1. Headshot
- **Filename:** `headshot.jpg`
- **Size:** 400x400px (square)
- **Usage:** About page sidebar
- **Description:** Professional photo of yourself

### 2. GameDev Learning Platform Screenshot
- **Filename:** `gamedev-screenshot.png`
- **Size:** 800x600px (or similar landscape)
- **Usage:** Project card and detail page
- **Description:** Screenshot of the LMS dashboard or course view

### 3. Portfolio Website Screenshot
- **Filename:** `portfolio-screenshot.png`
- **Size:** 800x600px (or similar landscape)
- **Usage:** Project card and detail page
- **Description:** Screenshot of this portfolio's home page

### 4. AI Pet Management Platform Screenshot
- **Filename:** `chatbot-screenshot.png`
- **Size:** 800x600px (or similar landscape)
- **Usage:** Project card and detail page
- **Description:** Screenshot of the pet app UI or chatbot interface

## Notes

- Images should be optimized for web (compressed, reasonable file size)
- PNG format recommended for screenshots
- JPG format recommended for the headshot
- If you don't have screenshots yet, the site will show letter placeholders
